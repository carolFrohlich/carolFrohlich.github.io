I selected a few of my favorite projects. You can check all projects on my Github page.

C-PAC

Brain Imagin processing software

The Configurable Pipeline for the Analysis of Connectomes is an open-source processing pipeline for resting state fMRI (functional Magnetic Resonance Imaging) data. C-PAC brings the power, flexibility and elegance of Nipype to users in a plug-and-play fashion; no programming required.

Open Cog Lab

Repository of tasks for assessing cognitive function

Open Cog Lab is a open source initiative for sharing computerized experiments. On this repository, you will find a variety of tasks for assessing cognitive function. Feel free to download our experiments and share your own!

Brainhack

Brain Imagin Hackaton Website

Brainhack is a hackathon that focuses on colaborative neuroimagin projects.I was one of the developers of the main website brainhack.org, and I also developed some websites for specific events, like Brainhack global 2017, Vienna 2016 and LA 2016.

Brain Orchestra

Music Generator

This project uses solely fMRI data to generate pleasant music. It uses basic mathematical principles of music to crunch brain data into music. You can read more about it on the publication: Generating music with resting-state fMRI data
