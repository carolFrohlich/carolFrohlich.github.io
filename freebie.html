Projects

✻

A selection of my projects. You can view more at Github .

C-PAC

Brain imaging data processing software

The Configurable Pipeline for the Analysis of Connectomes is an open source pipeline for high throughput processing of fMRI data using high performance and cloud computing.

Open Cog Lab

Repository of tasks for assessing cognitive function

Open Cog Lab is a repository of openly shared computerized experiments for assessing cognitive function (e.g. working memory, sustained attention) and neurofeedback.

Brainhack

Workshop website

Brainhack is a collaborative workshop for brain science. I helped develop the main website brainhack.org and websites for the specific events, (e.g., Brainhack global 2017, Vienna 2016 and LA 2016).

Brain Orchestra

Music generator

This project uses principles of music, unsupervised learning and signal processing to generate music from functional MRI data. It is described in the publication: Generating music with resting-state fMRI data
